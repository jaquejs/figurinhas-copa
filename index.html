Conectar ao Google Sheets

Os dados de serão salvos numa planilha Google Sheets compartilhada entre todos os usuários.

  1. Acesse console.cloud.google.com → crie um projeto.
  2. Ative a Google Sheets API e a Google Drive API.
  3. Em "Credenciais" crie um OAuth 2.0 Client ID tipo "Aplicativo da Web".
  4. Em "Origens JavaScript autorizadas" adicione a URL do seu GitHub Pages.
  5. Em "URIs de redirecionamento" adicione também a mesma URL.
  6. Em "Tela de consentimento" → "Audience" → publique o app.
